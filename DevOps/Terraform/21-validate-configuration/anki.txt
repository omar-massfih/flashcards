When does Terraform execute input variable validation?	Immediately, before generating a plan.
When does Terraform execute preconditions?	After generating a plan but before creating the associated resource, data source, or output.
When does Terraform execute postconditions?	After planning and applying changes to a resource, or after reading from a data source.
When does Terraform execute `check` blocks?	At the end of `plan` and `apply` operations.
What happens when a Terraform `check` assertion fails?	Terraform reports a warning and continues the current operation.
How do `check` blocks differ from variable validation and pre/postconditions (Terraform)?	Checks do not block operations when they fail.
What is continuous validation in HCP Terraform?	Regularly re-running checks, preconditions, and postconditions to verify infrastructure over time.
When does HCP Terraform continuous validation run `check` blocks?	Whenever workspace health assessments run, in addition to normal plan/apply execution.
