How do counts work with motions and operators (VS Code Vim)?	Prefix the command with a number to repeat it or extend its range, like `3w` or `2dd`.
What are operators in VS Code Vim?	Commands that act on text selected by a following motion or text object, like `dw`, `ciw`, or `yap`.
What does the delete operator `d` do (VS Code Vim)?	Deletes the text covered by the following motion or text object.
What does the yank operator `y` do (VS Code Vim)?	Copies the text covered by the following motion or text object.
What does the change operator `c` do (VS Code Vim)?	Deletes the target text and then enters Insert mode.
